# School Portal — Интеллектуальная образовательная экосистема

Инновационная платформа для управления учебным процессом, объединяющая учеников, учителей, родителей и администрацию школы в едином цифровом пространстве. Проект разработан для автоматизации школьной рутины и повышения качества образования за счет глубокой аналитики данных.

## Демо-версия доступна по ссылке: https://adiya-32.github.io/School-Portal/

## Основные возможности

### Для учеников
- **Интерактивный дневник:** Просмотр оценок в форматах FA (ФО), SAU (СОР) и SAT (СОЧ).
- **AI-Наставник:** Персональный помощник, анализирующий успеваемость и дающий советы по подготовке.
- **Рейтинговая система:** Мотивация через сравнение своих успехов с результатами класса и школы.
- **Лента новостей:** Все важные события и объявления школы в одном месте.

### Для учителей
- **Цифровой журнал:** Удобный ввод оценок с автоматическим расчетом процентов и итоговых баллов.
- **Зона риска:** Автоматическое выявление учеников, нуждающихся в дополнительном внимании.
- **Управление классами:** Быстрый доступ к спискам учеников и их статистике.

### Для родителей
- **Лента обновлений:** Мгновенные уведомления об успехах и достижениях ребенка.
- **Аналитика:** Наглядные графики прогресса по предметам и сравнение со средними показателями.
- **Прямая связь:** Возможность быстро связаться с учителями.

### Режим Киоска
- Специальный интерфейс для информационных панелей в холлах школы: дни рождения, герои недели и предстоящие мероприятия.

## Технологический стек

- **Frontend:** React 18 (Vite) — для высокой скорости работы и отзывчивости интерфейса.
- **Styling:** Tailwind CSS — современный подход к дизайну с поддержкой темной темы.
- **Animations:** Framer Motion — плавные переходы и живой интерфейс.
- **Backend:** Node.js + Express — масштабируемая серверная архитектура.
- **Icons:** Lucide React — лаконичная и современная иконографика.
- **State Management:** React Hooks (useState, useEffect) — эффективное управление данными.

## Быстрый старт

### Требования
- Node.js (версия 18 или выше)
- npm или yarn

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/adiya-32/School-Portal.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd School-Portal
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```

## Структура проекта

- `src/components/` — Переиспользуемые UI-компоненты.
- `src/services/` — Логика взаимодействия с API.
- `src/translations.ts` — Система мультиязычности (RU, KZ, EN).
- `server.ts` — Серверная часть приложения.
- `types.ts` — Описание структур данных и интерфейсов.
